# Circuit Problem Solver Lite V3

## 本次新增

- 支持撤销和重做操作，顶部按钮与 `Ctrl+Z` / `Ctrl+Y` 都可以使用
- 支持浏览器本地自动保存，刷新或重新打开页面后可恢复上次画布
- 支持一键恢复上次保存的画布
- 支持一键复制直流电路计算结果
- 支持下载 `circuit-solution.txt` 解题步骤文本
- 复制和下载内容会复用页面当前识别到的串联、并联或混联计算结果

Circuit Problem Solver Lite V3 是一个拼图式电路编辑器雏形。当前版本支持拖拽元件、点击端点建立连线，并把电路图保存为结构化 JSON 数据。

本版本仍然不做自动计算、不做复杂仿真，也不接入 AI API。

## 项目用途

这个项目用于练习使用 HTML、CSS 和 JavaScript 构建交互式电路编辑页面。用户可以把元件拖拽到网格画布上，再用左右端点建立连接，形成简单的电路图数据结构。

## 数据结构

程序内部使用 `components` 和 `connections` 保存电路图。

```js
{
  components: [
    {
      id: "R1",
      type: "resistor",
      value: "10Ω",
      x: 120,
      y: 72
    }
  ],
  connections: [
    {
      from: "V1.right",
      to: "R1.left"
    }
  ]
}
```

字段说明：

- `components`：所有元件
- `connections`：所有连线
- `id`：元件编号，例如 `R1`、`C1`、`V1`
- `type`：元件类型，例如 `resistor`、`capacitor`
- `type: "node"`：等电位点，不代表导线元件，导线保存在 `connections` 中
- 等电位点的连接端点为 `N1.center`
- `value`：元件显示值，例如 `10Ω`
- `x`：元件在画布中的横坐标
- `y`：元件在画布中的纵坐标
- `from`：连线起点，例如 `V1.right`
- `to`：连线终点，例如 `R1.left`
- `bendX` / `bendY`：可选字段，用于保存手动整理导线时的拐点位置

## 文件说明

- `index.html`：页面结构
- `style.css`：页面样式
- `script.js`：拖拽、编辑、删除、连线、中文说明、导入导出和 JSON 数据逻辑

## 使用方法

直接用浏览器打开 `index.html` 文件即可运行。

本项目不需要后端，不需要数据库，也不需要安装额外依赖。

点击“生成示例电路”可以自动创建一个由直流电源、电阻和等电位点组成的演示电路。

点击“生成并联示例”可以自动创建一个由直流电源、两个等电位点和两个并联电阻组成的演示电路。

点击“生成混联示例”可以自动创建一个由直流电源、一个串联电阻和两个并联电阻组成的演示电路。

点击“整理画布”可以把当前元件重新排列到整齐的网格位置，已有连线会自动跟随更新。

点击一根导线后，导线中间会出现一个小方块。拖动这个小方块可以手动整理导线路径，让线条绕开元件或看起来更整齐。

底部“电路结构分析”会根据当前元件和导线，自动显示元件数量、导线数量、未接入元件、等电位点数量、简单路径判断，以及每个元件的连接数量。

底部“直流串联计算”会在识别到一条简单路径，并且路径中包含 1 个直流电源和若干电阻时，自动计算：

- 串联总电阻：`R总 = R1 + R2 + ...`
- 总电流：`I = U / R总`
- 各电阻分压：`U = I × R`

当电路符合“1 个直流电源 + 2 个等电位点 + 多个电阻跨接在两个等电位点之间”的结构时，也会自动进行简单并联计算：

- 并联总电阻：`1 / R总 = 1 / R1 + 1 / R2 + ...`
- 总电流：`I总 = U / R总`
- 各支路电流：`I = U / R`

当电路符合“1 个直流电源 + 1 个串联电阻 + 1 组并联电阻”的结构时，会自动进行简单混联计算：

- 先算并联等效电阻：`1 / R并 = 1 / R2 + 1 / R3 + ...`
- 再算总电阻：`R总 = R串 + R并`
- 再算总电流：`I总 = U / R总`
- 再算串联电阻分压和各并联支路电流

## 暂不支持

- 完整识别串并联
- 更复杂的混联电路计算
- 电容、电感动态过程计算
- 图片识别
- AI 解题
- 电路仿真

## 后续升级方向

- 简单串并联识别
- 多级串并联混合电路识别与计算
- 更复杂的导线自动避让
