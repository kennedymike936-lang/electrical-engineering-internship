Circuit Problem Solver Lite V10 是一个拼图式电路编辑器。当前版本支持拖拽元件、点击端点建立连线、基础直流电路计算、虚拟万用表测量、红黑表笔测量和中文电路报告导出，并把电路图保存为结构化 JSON 数据。

本版本仍然不做复杂仿真，也不接入 AI API。

## 项目用途

这个项目用于练习使用 HTML、CSS 和 JavaScript 构建交互式电路编辑页面。用户可以把元件拖拽到网格画布上，再用左右端点建立连接，形成简单的电路图数据结构。

## 数据结构

程序内部使用 `components` 和 `connections` 保存电路图。

```js
{
  components: [
    {
      id: "R1",
      type: "resistor",
      value: "10Ω",
      rotation: 0,
      x: 120,
      y: 72
    }
  ],
  connections: [
    {
      from: "V1.right",
      to: "R1.left"
    }
  ]
}
```

字段说明：

- `components`：所有元件
- `connections`：所有连线
- `id`：元件编号，例如 `R1`、`C1`、`V1`
- `type`：元件类型，例如 `resistor`、`capacitor`
- `type: "node"`：等电位点，不代表导线元件，导线保存在 `connections` 中
- 等电位点的连接端点为 `N1.center`
- `value`：元件显示值，例如 `10Ω`
- `rotation`：元件旋转角度，支持 `0`、`90`、`180`、`270`
- `x`：元件在画布中的横坐标
- `y`：元件在画布中的纵坐标
- `from`：连线起点，例如 `V1.right`
- `to`：连线终点，例如 `R1.left`
- `bendX` / `bendY`：可选字段，用于保存手动整理导线时的拐点位置

## 使用方法

直接用浏览器打开 `index.html` 文件即可运行。

本项目不需要后端，不需要数据库，也不需要安装额外依赖。

点击“生成示例电路”可以自动创建一个由直流电源、电阻和等电位点组成的演示电路。

点击“生成并联示例”可以自动创建一个由直流电源、两个等电位点和两个并联电阻组成的演示电路。

点击“生成混联示例”可以自动创建一个由直流电源、一个串联电阻和两个并联电阻组成的演示电路。

点击“整理画布”可以把当前元件重新排列到整齐的网格位置，已有连线会自动跟随更新。

选中一个非等电位点元件后，点击“旋转 90°”可以改变元件方向；也可以按 `R` 快捷键旋转。

选中任意元件后，点击“复制元件”可以复制一个同类型、同数值、同旋转角度的新元件；也可以按 `Ctrl+D` 快捷键复制。

点击“缩小”“放大”“100%”可以调整画布视图比例。点击“适应视图”可以把当前电路自动放到画布中合适的位置。

点击“移动画布”后，拖动画布空白区域可以平移视图；再次点击“移动画布”可关闭平移模式。

点击“导出 SVG”可以下载当前电路图文件 `circuit-diagram.svg`，适合插入文档、报告或继续用矢量工具编辑。

点击“导出 PNG”可以下载当前电路图图片 `circuit-diagram.png`，适合直接放入作业、实验报告或分享给别人查看。

点击一根导线后，导线中间会出现一个小方块。拖动这个小方块可以手动整理导线路径，让线条绕开元件或看起来更整齐。

在“虚拟万用表”中点击“红笔”或“黑笔”，再点击画布端点，就可以放置对应表笔。红黑表笔都放好后，电压档会显示红笔到黑笔的电压差。

点击“复制报告”或“下载报告”可以生成一份中文文本报告，内容包括元件清单、连接关系、电路体检、计算步骤和当前万用表读数。

底部“电路结构分析”会根据当前元件和导线，自动显示元件数量、导线数量、未接入元件、等电位点数量、简单路径判断，以及每个元件的连接数量。

底部“直流串联计算”会在识别到一条简单路径，并且路径中包含 1 个直流电源和若干电阻时，自动计算：

- 串联总电阻：`R总 = R1 + R2 + ...`
- 总电流：`I = U / R总`
- 各电阻分压：`U = I × R`

当电路符合“1 个直流电源 + 2 个等电位点 + 多个电阻跨接在两个等电位点之间”的结构时，也会自动进行简单并联计算：

- 并联总电阻：`1 / R总 = 1 / R1 + 1 / R2 + ...`
- 总电流：`I总 = U / R总`
- 各支路电流：`I = U / R`

当电路符合“1 个直流电源 + 1 个串联电阻 + 1 组并联电阻”的结构时，会自动进行简单混联计算：

- 先算并联等效电阻：`1 / R并 = 1 / R2 + 1 / R3 + ...`
- 再算总电阻：`R总 = R串 + R并`
- 再算总电流：`I总 = U / R总`
- 再算串联电阻分压和各并联支路电流

## 暂不支持

- 完整识别串并联
- 更复杂的混联电路计算
- 电容、电感动态过程计算
- 图片识别
- AI 解题
- 电路仿真
