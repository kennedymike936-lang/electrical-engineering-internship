# Circuit Problem Solver Lite

Circuit Problem Solver Lite 是一个纯前端电路解题小工具。第一版只支持文本输入，不识别图片，主要用于计算纯电阻直流电路中的串联和并联问题。

## 项目用途

这个项目适合作为电气工程学习和实习记录中的小练习项目，用来帮助理解：

- 等效电阻
- 欧姆定律
- 串联电路电流、电压关系
- 并联电路电压、电流关系
- 中文解题步骤的自动生成

## 功能

- 输入电源电压
- 输入多个电阻值
- 选择连接方式：串联或并联
- 计算等效电阻
- 计算总电流
- 计算每个电阻的电压
- 计算每个电阻的电流
- 自动生成中文解题步骤

## 支持的输入格式

电阻值使用英文逗号分隔：

```text
10, 20, 30
```

## 使用公式

### 串联电路

等效电阻：

```text
Req = R1 + R2 + R3 + ...
```

总电流：

```text
I = U / Req
```

各电阻电流：

```text
I1 = I2 = I3 = I
```

各电阻电压：

```text
Ui = I × Ri
```

### 并联电路

等效电阻：

```text
1 / Req = 1 / R1 + 1 / R2 + 1 / R3 + ...
```

总电流：

```text
I = U / Req
```

各电阻电压：

```text
U1 = U2 = U3 = U
```

各支路电流：

```text
Ii = U / Ri
```

本项目不需要后端，不需要数据库，也不需要安装额外依赖。
